# QuotaLab · Inadimplência PRO

Central de decisão e simulador de inadimplência condominial.

## Estrutura

```
quotalab-deploy/
├── ql_inadimplencia_SALVO__3_.html   ← app principal (single-file)
├── manifest.json                      ← PWA manifest
├── README.md
└── icons/
    ├── icon.svg                       ← favicon vetorial (browsers modernos)
    ├── favicon.ico                    ← favicon clássico (16/32/48px)
    ├── apple-touch-icon.png           ← ícone iOS (180×180)
    ├── icon-16.png
    ├── icon-32.png
    ├── icon-48.png
    ├── icon-64.png
    ├── icon-128.png
    ├── icon-192.png                   ← PWA Android
    └── icon-512.png                   ← PWA splash / store
```

## Deploy

### GitHub Pages
1. Suba esta pasta para um repositório GitHub
2. Em Settings → Pages, selecione a branch e a raiz `/`
3. O app estará disponível em `https://seu-usuario.github.io/seu-repo/ql_inadimplencia_SALVO__3_.html`

### Uso local
Abra `ql_inadimplencia_SALVO__3_.html` diretamente no navegador — não precisa de servidor.

### Instalar como PWA
No Chrome/Edge: abra o app, clique no ícone de instalação na barra de endereço.
No iOS Safari: compartilhar → "Adicionar à Tela de Início".
